Process

How a prediction works

1

Image Upload

You upload a JPG, PNG, or WEBP photo. Flask receives it, saves it temporarily, and passes the file path to the model pipeline.

2

Preprocessing

The image is resized to 160×160 pixels and normalized using MobileNetV2's preprocessing function — scaling pixel values to the range [-1, 1] that the model expects.

3

Feature Extraction

Three convolutional layers progressively extract features from your image — edges and textures first, then shapes, then high-level patterns like fur or snouts.

4

Classification

A small trainable head (GlobalAveragePooling → Dropout → Dense with sigmoid) outputs a single number between 0 and 1. Below 0.5 → Cat. Above 0.5 → Dog.

5

Result Display

Flask sends the label and confidence score back to the browser. The result is also saved to prediction history so you can review past classifications.
