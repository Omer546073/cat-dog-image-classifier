🐾 PawDetect — Cat vs Dog Image Classifier

A beginner-friendly machine learning web app built with Flask and TensorFlow.
Upload any image and the model instantly predicts whether it's a cat or a dog,
with a confidence score. Includes prediction history, an about page explaining
how the model works, and a clean dark-themed UI.

Built with: Python · Flask · TensorFlow · Keras · Custom CNN · HTML/CSS
Dataset: Cats vs Dogs (Kaggle)
